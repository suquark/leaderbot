Install
=======

Install with ``pip`` from `PyPI <https://pypi.org/project/leaderbot/>`_:

.. prompt:: bash
    
    pip install leaderbot

Alternatively, clone the source code and install with

.. prompt:: bash
   
    cd source_dir
    pip install .
